Thor daily chores, o que é?

Thor daily chores é um aplicativo desenvolvido para listar as tarefas de nossos colaboradores e organizar os afazeres do dia a dia. O aplicativo conta com um sistema intuitivo de adição, edição e exclusão das tarefas, contempla uma descrição com data, hora e se a tarefa já foi realizada ou não. As tecnologias usadas para desenvolver são: HTML, CSS, JavaScript e Banco de dados LS.
